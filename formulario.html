Tipos de Formulario

Text: Campo para entrada de texto simples.

Email: Campo específico para endereços de e-mail. O navegador valida automaticamente o formato do e-mail.

Password: Campo para senha. Os caracteres digitados são mascarados por asteriscos ou pontos.

Number: Campo para entrada de números.

Date: Campo para seleção de datas.

Textarea: Área de texto para entrada de várias linhas.

Radio: Botões de opção. Apenas um pode ser selecionado por vez.

Checkbox: Caixas de seleção. Múltiplas opções podem ser selecionadas.


Ler
Jogar

Select: Lista suspensa ou caixa de seleção.


Tipos de Botões em Formulários HTML

Função: Envia os dados do formulário para o servidor, geralmente para processamento.

Função: Limpa todos os campos do formulário, retornando-os ao seu estado inicial.

Função: Cria um botão que pode executar qualquer ação personalizada, como chamar uma função JavaScript.
