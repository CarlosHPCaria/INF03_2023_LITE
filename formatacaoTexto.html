Formatação de Texto em HTML

< b > e < strong >: Ambas tornam o texto negrito. A diferença está na semântica:

  • < strong > indica que o texto é importante ou urgente.
  • < b > serve apenas para destacar visualmente.
Ex: Este texto é importante e merece destaque.
Ex: Esssa palavra "cadeira" merece destaque

< i > e < em : Ambas tornam o texto itálico. A diferença está na semântica:

  • < em > indica ênfase em um texto.
  • < i > é mais utilizado para nomes científicos, termos estrangeiros ou pensamentos.
Ex: A palavra amor é um sentimento universal.
Ex: "Manihot esculenta é o nome científico da mandioca, também chamada de macaxeira e aipim"

< u >: Sublinha o texto

Ex: Este texto está sublinhado.

< sup >: Coloca o texto em sobrescrito.

Ex: H₂O é a fórmula da água.

< sub >: Coloca o texto em subscrito.

Ex: x² + 2x + 1 = 0

< ins >: Indica texto inserido.

Ex: O gato preto subiu na árvore.

< del >: Indica texto deletado.

Ex: Este texto será removido.

< small >: Diminui o tamanho da fonte.

Ex: Este texto é menor.

< mark >: Marca um texto com um fundo colorido, geralmente amarelo, para destacar.

Ex: Procure a palavra chave neste texto.

< pre >: Preserva os espaços em branco e quebras de linha, ideal para exibir código-fonte.

Ex: 
          function hello() {
          console.log("Hello, world!");
        }

< blockquote >: Cria uma citação longa, geralmente com recuo.

Ex:
"A única coisa que nunca falha é a mudança." - Heráclito

< q >: Cria uma citação curta.

Ex: Ele disse: Vou pensar sobre isso.
